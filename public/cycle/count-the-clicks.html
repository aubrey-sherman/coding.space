Count the clicks

(If any of the below previews are blank, right-click them and click "Reload Frame".)

In this project, we're going to build a button that counts how many times it's been clicked.

1) From the Text Elements menu, drag a heading to the page. Change the text to give your project a name.

2) From the Layout menu, drag "arrange sub-elements" into the page to center everything.

3) From the Input Elements menu, drag a button element onto the page. Click on it and see what happens.

4) Change the "arrange sub-elements" blocks to "top to bottom".

5) In the Variable menu, create a variable called "clicks". Set clicks to 0 in your page.

6) From the variable menu, drag the clicks variable in to replace the button's text.

7) Instead of printing a message, make the clicks variable go up by one when the button is clicked.

8) Add more text into the button to make it say the full sentance, "I have been clicked X times", where X is the value of counts. (You may need horizontal spacers from the Container Elements menu.)

9) From the Layout menu, drag the background color block inside of the button.

10) Drag "test if true if false" in Logic to the background color block in the button. If the number of clicks is even, then the background should be green, and if the number of clicks is odd then the background color should be red.
