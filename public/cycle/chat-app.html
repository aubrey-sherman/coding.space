Chat App

(If any of the below previews are blank, right-click them and click "Reload Frame".)

In this project, we're going to build a real-time chat app to talk with your friends.

1) Create an account on firebase.google.com and create a new project.
2) Change the Rules on the database by clicking on "Database" then "RULES", then "LEARN MORE" in the Console.





Then choose "PUBLIC" from the example rule sets.



Replace the rules for your project with the PUBLIC rules. This will allow your game code to read and write freely to the database.
3) Go to the Firebase console, choose "Add Firebase to your web app."



Firebase will generate the config code for your project.

From the Firebase menu, drag the blocks to set up Firebase into the page. Copy and paste the config values one by one from Firebase into your project.

5) Put a heading on the screen to give your app a title.

6) From the Layout menu, drag "arrange sub-elements" into the page to center everything.

7) Put an input box on the screen.

8) Change the "arrange sub-elements" block so that the title is at the top of the page and the input box is at the bottom.

9) When the enter key is pressed in the input box, add the input1Text to a list "messages" in firebase. Look in your firebase database to see if the message is there.

10) When the enter key is pressed, set the input1Text to the an empty string.

11) Now we are going to retrieve the messages from the database to display. Create a variable "list" that will contain the messages. Set list to an empty list.

12) Below the block that sets up firebase, get the "messages" list. "Every time" "messages" changes, print the firebaseValue. The "to json" block in Text will help. (Right click the white window and select "reload this frame" to see what it should look like.)

13) Set the list variable to the values (you can get the values of an object with a block from Objects) of the firebaseValue. (Right click the white window and select "reload this frame" to see what it should look like.)

14) Using a for each in list block (in Control), display all of the messages.

15) Style the messagese using font align center, border radius, and background color.
